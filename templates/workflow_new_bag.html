QR codes: stations vs bags

Shipment bags — Bags are created when inventory is received (shipment/receiving). Use the selects below to pick the physical bag, then the product that matches that bag’s tablet type. The system assigns the next idle QR card to track that bag through blister → sealing → packaging.

Station — Each sealing station has a QR whose URL is /workflow/station/<station_scan_token>. Scanning it opens the floor station page for that station. You can also paste the station token on manual entry.

Bag / card — The physical card’s scan_token identifies the workflow bag on the station page together with the station. Station QR and card token serve different roles: station = which machine UI; card = which bag is in progress.
