About floor stations (blister, sealing, packaging)

All of these use the same workflow_stations table: each row is one QR / URL (/workflow/station/<scan_token>) that opens the floor tablet UI. The Station type only organizes the admin view and controls which production machine role you can link (sealing vs blister; packaging can reference any machine for your own tracking). Physical bags still use the QR cards table below.
